# 🛡️ Kavach AI – Cyber Safety Companion

## 🚀 Overview

Kavach AI is an AI-powered platform designed to detect and prevent cyberbullying.
It helps victims understand harmful content, provides emotional support, and guides them with actionable steps.

---

## 🎯 Problem

Cyberbullying is a growing issue among students.
Many victims don’t know:

* Whether they are being bullied
* What actions to take
* How to respond safely

---

## 💡 Solution

Kavach AI analyzes messages and screenshots to:

* Detect cyberbullying
* Classify severity
* Identify type (threat, harassment, photo misuse)
* Provide emotional support
* Suggest legal and safety actions

---

## ✨ Features

* 🔍 Cyberbullying Detection
* 📊 Severity Analysis
* 💬 Emotional Support
* 📝 Complaint Generator
* 🆘 SOS Alert System
* 📸 Screenshot Analysis (OCR)

---

## 🛠️ Tech Stack
